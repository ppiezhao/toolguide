# TabBar 图标说明

请在此目录添加以下 8 张 81x81px 的 PNG 图片：

1. **home.png** - 首页未选中图标
2. **home-active.png** - 首页选中图标（绿色）
3. **discover.png** - 发现未选中图标
4. **discover-active.png** - 发现选中图标
5. **star.png** - 收藏未选中图标
6. **star-active.png** - 收藏选中图标
7. **user.png** - 我的未选中图标
8. **user-active.png** - 我的选中图标

**图标风格建议：**
- 未选中：灰色 (#999999)
- 选中：微信绿 (#07C160)
- 建议使用线性图标风格

**图标来源：**
- 可使用 IconFont、Figma 等设计工具
- 或使用在线图标库（如 IconPark）
- 推荐 24px 线性图标
